# CoolRocks
Client for the PeculiarPlaces API

## Requirements

- [Node.js](https://nodejs.org/) 20+ (for local development)
- [Docker](https://www.docker.com/) (for containerised builds)

## External Libraries

| Library | Version | Purpose |
|---|---|---|
| [React](https://react.dev/) | ^18.3.1 | UI framework |
| [react-dom](https://react.dev/) | ^18.3.1 | React DOM rendering |
| [Leaflet](https://leafletjs.com/) | ^1.9.4 | Interactive maps |
| [react-leaflet](https://react-leaflet.js.org/) | ^4.2.1 | React wrapper for Leaflet |
